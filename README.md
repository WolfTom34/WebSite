# SAFE VALLEY - SVE | Orbital Operating System

Advanced Autonomous Robotics & Defense Systems. An immersive, spatial web terminal for the Safe Valley Engineering - SVE.

## Technical Architecture
This project follows **Hexagonal Architecture** principles, separating core domain logic from infrastructure concerns (React/Three.js).

### Core Features
- **Neural Lattice**: High-density 3D visualization of orbital node networks.
- **Tactical HUD**: Fully interactive command deck with real-time telemetry.
- **Dynamic Localization**: Persistent multi-language support (English/French).
- **Performance Tiering**: High-efficiency rendering engine with ULTRA/HIGH/ECO tiers.

## Getting Started
### Prerequisites
- Node.js (v18+)
- NPM

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build & Deploy
```bash
npm run build
./deploy.sh
```
